\centerline{\bf ENIAC Simulator Command Reference}
\vskip 12pt
\noindent
{\bf Commands}

{\advance\leftskip 0.5in
\item{\tt b:} Press a button
\item{\tt d:} Display a unit status
\item{\tt D:} Display all unit statuses
\item{\tt f:} Associate a file with the punch or reader
\item{\tt l:} Load a configuration file
\item{\tt p:} Plug in a jumper
\item{\tt q:} Quit
\item{\tt r:} Reset a single unit
\item{\tt R:} Reset all units
\item{\tt s:} Set a switch setting
\item{\tt u:} Declare units being used
\item{\tt dt:} Declare the number of data trunks in use
\item{\tt pt:} Declare the number of program trunks in use

}
